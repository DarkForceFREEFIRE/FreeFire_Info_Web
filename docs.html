Free Fire Public Information API

v1.0 (Stable)

Welcome to Walker's public API network for Garena Free Fire. This service enables developers to query player states, anti-cheat status, and guild parameters.

Rate Limiting Policy: Rate limited to 60 transactions per minute per originating IP addresses to guarantee responsive performance.

Base API URL

https://wzapiinfo.vercel.app
GET /get

Retrieve detailed game attributes, active inventory equipment, rank thresholds, and clan metadata of the specified target.

Required Parameters

Parameter Data Type Required Description
uid String Yes Numeric client player UID. Example: 10597688191
region String No Target zone (e.g. SG, BR, IND). Defaults to automated searches.

Transaction Sample

curl -X GET "https://wzapiinfo.vercel.app/get?uid=10597688191"
GET /check_ban

Inspect whether the targeted player UID is currently blacklisted or flagged as suspended by Garena anti-cheat systems.

Required Parameters

Parameter Data Type Required Description
uid String Yes Target numeric account UID.

Transaction Sample

curl -X GET "https://wzapiinfo.vercel.app/check_ban?uid=10597688191"
